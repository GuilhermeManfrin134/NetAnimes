<h1 align="center">NetAnimes</h1>
<p align="center">Desafio proposto no Bootcamp HTML Web Developer da Digital Innovation One. </p>
<p align="center">Consistia em criar uma Interface de Streaming com base na aula de "Recriando a Interface do Netflix". </p>
<p align="center">Desenvolvido com a utilização de HTML, CSS, JavaScript e JQuery.</p>

<h2 id="Anime">Anime</h2>

<p>Em cada Card, assim que se passa o mouse por cima é possível ver o nome da obra, e dois botões.</p>
<img src="./img/AnimeCard.gif" style="width: 600px">
<p>O botão de informações a esquerda ti redireciona para uma página do Anime no <a href="https://anilist.co/home">Anilist</a>, site de catalogação de Animes e Personagens.</p>
<img src="./img/Info.gif" style="width: 600px">
<p>A direita, o botão "play" o levará para um link do <a href="https://www.youtube.com/">YouTube</a> contendo a música de abertura do anime em questão.</p>
<img src="./img/Play.gif" style="width: 600px">
